# ☕ Bark and Beam Coffee (BnB) Landing Page

A modern landing page website for **Bark and Beam Coffee (BnB)**, built to elevate brand presence and attract customers through a visually engaging and high-performance web experience.  
This project focuses on delivering a premium UI/UX, smooth animations, and SEO optimization to maximize reach and user engagement.

## 🚀 Features
- ✨ Modern UI design
- ⚡ Fast and optimized performance
- 📱 Fully responsive (mobile-first)
- 🎬 Smooth animations and interactions
- 🔍 SEO-friendly structure
- 🧭 Clean and scalable code architecture
---
## 🛠️ Tech Stack
- **Next.js** – React framework for production-ready apps  
- **Tailwind CSS** – Utility-first CSS framework  
- **Framer Motion** – Animation library for smooth interactions  
- **Vercel** – Deployment and hosting platform
